# рџЋ“ Smart Campus Operations Hub (PAF 2026 Group Project)

A full-stack university operations platform to manage **campus resources**, **facility bookings**, **maintenance tickets**, and **role-based access**, with secure authentication and real-time notifications.

---

# рџ“Њ Project Overview

The **Smart Campus Operations Hub** centralizes university operations into one system:

* Students can **browse facilities**, **request bookings**, and **report issues**
* Admins can **manage resources**, **approve bookings**, and **monitor system activity**
* Technicians can **handle maintenance tickets and updates**

The system is built using a **React frontend** and a **Spring Boot REST API backend**, secured with **JWT authentication** and optional **OAuth2 login**.

---
